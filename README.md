## 📁 Como baixar o projeto

```bash

# Clonar o repositório
$ git clone https://github.com/Dani2003G/Projeto-Anna-Bella

# Entrar no diretório
$ cd Projeto-Anna-Bella

# Instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start

```
